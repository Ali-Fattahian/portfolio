PROJECTS

Prime Connect is an e-commerce website with features such as: usage of blurhash for user experience improvement, fully separate admin panel with analysis of sales, jwt auth system and advanced filtering for products section.

Twitter clone is a full stack website built with Django, DRF and React. It has: Email validation, explore system based on tweets and users, user suggestion system, CRUD operations for tweets and profiles and reply, like, bookmark system.

Instagram clone is a website which utilizes Django for showing templates. Features are: Email validation, search system, follow system for users, secure password reset system. CRUD operations, like, comment and bookmark system for posts.

Techie Tweed is a blog focused on technical articles. Its features are: different groups and permissions system for users, CRUD operations and pagination for posts, weekly automatic backup system, session based authentication and bookmark system.
